## üìã Detailed Requirements

### Benefits Section

**HTML Structure:**
```html
<section class="pls-benefits-section">
    <h2 class="pls-section-title">The Benefits</h2>
    <div class="pls-benefits-grid">
        <div class="pls-benefit-card">
            <div class="pls-benefit-icon">
                <img src="..." alt="..." />
            </div>
            <div class="pls-benefit-label">Benefit Name</div>
        </div>
        <!-- More benefit cards -->
    </div>
</section>
```

**CSS Requirements:**
- Grid layout: `grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))`
- Card styling with hover effects
- Icon/image support
- Responsive design
- Professional spacing

**Data Structure:**
- `benefits_json` should support:
  ```json
  [
    {
      "label": "Extreme hydration",
      "icon": "https://...",
      "image": "https://..."
    }
  ]
  ```

### Key Ingredients Section

**HTML Structure:**
```html
<section class="pls-key-ingredients-section">
    <h2 class="pls-section-title">Key Ingredients</h2>
    <div class="pls-ingredients-key-grid">
        <div class="pls-ingredient-card pls-ingredient-card--key">
            <div class="pls-ingredient-image">
                <img src="..." alt="..." />
            </div>
            <div class="pls-ingredient-name">Ingredient Name</div>
        </div>
        <!-- More ingredient cards -->
    </div>
</section>
```

**CSS Requirements:**
- Grid layout matching reference
- Image support
- Card styling
- Responsive design
- Professional spacing

**Data Structure:**
- Ingredient taxonomy should support image meta
- `key_ingredients_json` should reference ingredient IDs
- Images loaded from taxonomy meta or ingredient data

### Layout & Typography
